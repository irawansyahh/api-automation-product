Installation & Setup
Prerequisites
	•	Java 17 or higher
	•	Maven 3.6+
	•	Your favorite IDE (IntelliJ IDEA, Eclipse, or VS Code)

Step 1: Clone and Setup
git clone <repository-url>
cd api-automation-product

java -version
mvn -version

Step 2: Install Dependencies
mvn clean install

Run all test
mvn test
